About me

Hi. My name is Dave. I am a Python software engineer living in Bristol. I have 4 years experience developing and maintaining python code for test instrumentation which decodes and displays ethernet traffic. This was part of my role as a trials and integration engineer working on aircraft, uavs and ships. I am now looking for my first full time role as a software developer.

I recently completed a Udemy Python course to improve my Python skills (Andre Neagoie Zero to Mastery 2021). This included 37 hours of lectures plus coursework. You can see my Github here. My projects included machine learning and data science demos (Pandas/Matplotlib/Jupyter Notebooks), an image recognition tool (Imageai), a website (Flask Server), a secure password checker (Pwned), Twitter bot (Tweepy API), web scraper (BeautifulSoup4), unit testing (Pytest) and automated testing demo (Selenium WebDriver).
